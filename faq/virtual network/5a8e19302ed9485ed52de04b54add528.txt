是否可以在 VNet 中使用 Azure 应用服务 Web 应用？\n\n是的。 可以使用 ASE（应用服务环境）在 VNet 中部署 Web 应用。 如果为 VNet 配置了点到站点连接，Web 应用可以安全地连接和访问 Azure VNet 中的资源。 有关详细信息，请参阅以下文章：\n\n*   [将应用与 Azure 虚拟网络进行集成](https://docs.azure.cn/app-service-web/web-sites-integrate-with-vnet)\n*   [将 VNet 集成和混合连接用于 Web 应用](https://docs.azure.cn/app-service-web/web-sites-integrate-with-vnet)
